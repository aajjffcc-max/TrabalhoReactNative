# MyApp — Componentes 

Trabalho acadêmico: criação de 3 componentes reutilizáveis e uso do StyleSheet no React Native usando Expo.

## 📁 Estrutura do Projeto

```
MyApp/
├── components/
│   ├── Header.js     
│   ├── Footer.js     
│   └── Botao.js      
├── App.js            
├── app.json
├── babel.config.js
├── package.json
└── .gitignore
```

## ▶️ Como rodar o projeto

### 1. Instalar dependências
```bash
npm install
```

### 2. Iniciar o projeto
```bash
npx expo start
```

## 🧩 Componentes

| Componente | Props |
|------------|-------|
| `Header`   | `titulo`| 
| `Footer`   | `texto` | 
| `Botao`    | `label`, `cor`|
